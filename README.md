### **Installation**

```bash
# 1. Clone the repository
git clone https://github.com/Prajwal18py/SMART-CSV-HEALTH-CHECKER.git

# 2. Navigate to directory

# 4. Activate virtual environment

# 6. Run the app
streamlit run app.py
```
